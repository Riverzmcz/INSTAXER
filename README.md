## 📖 项目简介 / Introduction

**INSTAXER** 是一款追求极简主义美学的在线富士拍立得生成器。它可以将你的数码照片转化为具有真实胶片质感的拍立得相纸。支持多种画幅选择及深度的色彩微调，所有的图像处理均在本地浏览器完成，保护您的隐私。

**INSTAXER** is a minimalist web-based Fujifilm Instax generator. It transforms your digital photos into authentic film-style prints. With support for multiple formats and deep color adjustments, all processing happens locally in your browser to ensure maximum privacy.

---

## ✨ 核心功能 / Key Features

* **经典画幅 (Multi-Format Support):** * `MINI` (3:4) | `SQUARE` (1:1) | `WIDE` (4:3)
* **富士色彩实验室 (Film Lab Filters):**
    * **曝光与色温 (Exposure & Warmth):** 模拟物理相纸的感光宽容度。
    * **经典青调 (Fuji Blue):** 还原富士胶片标志性的阴影青蓝色调。
    * **胶片颗粒 (Grain):** 增加银盐质感，让数码照片更具“厚度”。
* **光学模拟 (Optical Effects):**
    * **暗角 (Vignette):** 模拟镜头边缘失光，增强氛围感。
    * **随机漏光 (Random Light Leak):** 每一张都是独一无二的化学反应。
* **交互体验 (UX):** 支持手势/鼠标缩放、平移裁剪，一键导出高清 PNG。

## 🛠️ 技术实现 / Tech Stack

* **HTML5 Canvas:** 高性能图像渲染与合成。
* **Vanilla JavaScript:** 无框架依赖，轻量且快速。
* **CSS3 Filter & Mix-blend-mode:** 模拟真实胶片叠色与纹理。

## 🚀 如何使用 / How to Use

1.  **Select:** 选择你喜欢的相纸规格。
2.  **Upload:** 点击区域上传照片，并拖拽/缩放至理想构图。
3.  **Adjust:** 在下方控制面板微调滤镜与光学效果。
4.  **Export:** 点击“生成并保存”，获取你的专属拍立得。

## 🔒 隐私声明 / Privacy

本项目不设服务器上传功能。您的所有照片仅在您的浏览器内存中处理，离开页面后即刻销毁，我们不会收集任何个人影像数据。

No images are uploaded to any server. All processing is done locally in your browser to ensure your privacy.
